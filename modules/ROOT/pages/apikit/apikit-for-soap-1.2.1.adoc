== Issues Fixed in this Release

* Fixed error when SOAP Router was impacting significantly on the response time of the application.
* Fixed error when the stream is closed when working with a big payload.
* Fixed error when `errorDescription` was unable to propagate from subflow using APIKit for SOAP.
