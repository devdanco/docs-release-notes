== 2.0.1

*June 26, 2019*

=== Minimum Mule Version

*Mule 4.1.1*

=== New Features and Enhancements

None.

=== Fixed Issues

* JSON Module throws NPE when the connector can't load the schema. (https://www.mulesoft.org/jira/browse/MULE-15530[MULE-15530])
* Update Guava version to 27.1-jre. (https://www.mulesoft.org/jira/browse/MULE-16648[MULE-16648])
* Upgrade Jackson to 2.9.9 in all branches. (https://www.mulesoft.org/jira/browse/MULE-17021[MULE-17021])
 
== 2.0.0

*July 5, 2018*

=== Minimum Mule Version

*Mule 4.1.0*

=== New Features and Enhancements

Errors are no longer reported as a List of Strings. Errors are now reported as an actual JSON document. See xref:connectors::json/json-schema-validation.adoc[validating documents against a JSON schema with the JSON module].

=== Fixed Issues

* JSON Schema Validator does not flag errors in identical JSON objects. (https://www.mulesoft.org/jira/browse/MULE-15303[MULE-15303])

== 1.0.1

*May 7, 2018*

=== Minimum Mule Version

*Mule 4.1.0*

=== New Features and Enhancements

* Input metadata is more accurate. The connector used to communicate that required
any kind of structure. Now, it is more accurate and shows that it requires any *JSON* structure.
* Upgraded Jackson version due to https://nvd.nist.gov/vuln/detail/CVE-2018-7489[the CVE-2018-7489 vulnerability].

=== Fixed Issues

* Stream consumed when static input resolver is used. (https://www.mulesoft.org/jira/browse/MULE-14753[MULE-14753])
* Upgrade Jackson to 2.9.5. (https://www.mulesoft.org/jira/browse/MULE-14795[MULE-14795])
