== Hardware and software requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* *JDK*: JRE 1.7.0 (recommended JRE 1.7.0_79/80), JRE 1.8
* *Maven*: Maven 3.3, Maven 3.5
